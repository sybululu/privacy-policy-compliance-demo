# 隐私政策合规审查系统

基于浙大CAPP-130 AI模型的隐私政策合规智能审查系统。

## 功能
- 输入隐私政策文本
- 自动识别12项违规指示符
- 生成合规评分和风险等级
- 提供整改建议

## 技术栈
- 前端：Gradio
- 后端：FastAPI + 浙大CAPP-130模型
- 模型：PERT (风险识别) + mT5 (整改生成)
